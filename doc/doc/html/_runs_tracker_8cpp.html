                 Ralph M. Parsons Laboratory
            Massachusetts Institute of Technology


                    Beta Release, 9/2001

RunTracker.cpp: Utility Program for tRIBS used to store run information of the input *.in file and RUN_NAME_ID when using the Model Option -O

Program compiled separately from tRIBS as: UNIX% CC -o <executable> RunsTracker.cpp Example: CC -o ../tracker RunsTracker.cpp (inside Utilities dir)

To compile the program on Tru64 (Alpha DEC Station): parana:~ cxx -o <executable> RunsTracker.cpp -model ansi

Note: RunsTracker must be run within *.in Directory Example: ../tRIBS/tracker *.in runID Alternatively, the tracker can be run using hill_tracker file
