Detailed Description

         tRIBS Distributed Hydrologic Model

        TIN-based Real-time Integrated Basin Simulator
         Ralph M. Parsons Laboratory
        Massachusetts Institute of Technology

tHydroMet.h: Header file for class tHydroMet

This class encapsulates the hydrometeorological from a weather station. Functions provided to create and access micrometeorological measurements. Data members include weather data and station information.
