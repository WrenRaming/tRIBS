tInputFile::ReadItem( const int &datType, const char *itemCode )

Reads one parameter from the file. The format is assumed to be a line of text that begins with the keyword, followed by a line containing the parameter to be read. The function is overloaded according to the type of data desired. Arbitrary order of the items in the infile allowed. Routine searches through list until it finds itemCode.
