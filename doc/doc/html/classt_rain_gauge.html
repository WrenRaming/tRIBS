         tRIBS Distributed Hydrologic Model

        Massachusetts Institute of Technology

tRainGauge.h: Header file for class tRainGauge. Modeled after tHydroMet

This class encapsulates the rainfall data from a raingauge station. Functions provided to create and access measurements. Data members include rainfall data and station information.
