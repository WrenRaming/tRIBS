tKinemat::KinematWave()

Implements kinematic wave routing model for a stream reach with fixed upper boundary condition Hupp for a stream reach initially having water level HLev(his) and lateral influx in the nodes Reff. The geometry of the channel is "contained" in arrays C, Y1, Y2, Y3 which are the coefficient vectors computed as function of channel widths, lengths, slopes, roughness.

The function updates the level vector HLev and returns the status of the update in 'check'

It is assumed that the following variables are "known" to the function:

  • n, m, m1
